## Getting Started

### Prerequisites
- Node.js 18+ installed

### Installation

```bash
npm install --legacy-peer-deps
```

### Run locally

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for production

```bash
npm run build
```

Output is generated in the `dist/` folder.

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
